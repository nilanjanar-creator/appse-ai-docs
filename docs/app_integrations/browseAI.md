Browse AI is the easiest way to extract and monitor data from any website. It allows you to turn any website into a self-filling spreadsheet or a custom API without writing any code. Integrating Browse AI into appse ai enables you to automate web data extraction by triggering your pre-trained Robots on demand.

---

## Set Up Credential

:::info

Before you create a credential for Browse AI using appse ai, ensure you have a Browse AI account and have retrieved your API Key from the dashboard settings.

:::

### Required Fields

You'll be asked to fill in the following details:

| Field           | Description                                        |
| --------------- | -------------------------------------------------- |
| Connection Name | A name to help you identify this connection        |
| API Key         | Your API Key from the Browse AI dashboard settings |

---

### Step-by-Step Guide

#### 1. Open the Credential Form

Click **Select a Credential** and choose **Browse AI** from the application list.

<img src="/img/credentials/browseAI/browse-ai-setup-new-credi-appseAI.png" alt="appse ai Browse AI Select Credential" width="700"/>



<br/>

This opens the Browse AI credential form. Add your **Connection Name**.

#### 2. Sign In to the Browse AI Dashboard

Go to [Browse AI](https://www.browse.ai/) and sign in to your account.




#### 3. Get Your API Key

After logging in, click on your profile icon or navigate to **Account Settings** and then the **API** tab.

<img src="/img/credentials/browseAI/click-create-api-key-browseAI.png" alt="appse ai Browse AI Get API Key" width="700"/>


<img src="/img/credentials/browseAI/enter-apikey-name-browse-ai.png" alt="appse ai Browse AI Get API Key" width="700"/>

<img src="/img/credentials/browseAI/copy-apikey-browseAI.png" alt="appse ai Browse AI Get API Key" width="700"/>

Copy your **API Key**.

#### 4. Paste API Key in appse ai

Return to the appse ai credential form. Paste the copied API key into the **API Key** field and click **"Save"** to store and validate your credential.
<img src="/img/credentials/browseAI/click-save-browse-Ai.png" alt="appse ai Browse AI Get API Key" width="700"/>
<br/>

:::warning

Keep your API key secure. Your API key grants access to your robots and credits. If your key is compromised, regenerate it immediately from the Browse AI dashboard.

:::

## Actions

Here is the available action for Browse AI:

### Actions

- **Run Robot** — Trigger a specific Robot to start a new task. You can provide an **Origin URL** to override the default starting page and pass **Input Parameters** for dynamic data extraction (e.g., search keywords or specific product IDs).
