# Update an Existing Credential

Keeping credentials up to date is crucial for maintaining uninterrupted data flow and preventing connection errors in your automation processes.

## How to Update an Existing Credential

To update an existing credential in **appse ai**, follow these simple steps:

1. **Open the three-dot menu** beside the credential you want to update. And click the edit option.

<img src="\img\platform\key-concepts\credentials\update-credential\edit-option.png" alt="appse ai find credential edit" width="700"/>

2. A **Configure Credential** pop-up will appear, allowing you to modify and update the existing details.

<img src="\img\platform\key-concepts\credentials\update-credential\credential-popup.png" alt="appse ai find credential update popup" width="700"/>

3. When finished, choose **Save & Authorize** to apply the changes.
