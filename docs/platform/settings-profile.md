This guide explains the features of the **Profile Settings** page in APPSe AI and how users can interact with them to update their personal information.

---

#### Where to Find This:

Navigate through the left sidebar: `Settings` > `Profile`

---

### 1. Personal Information Section

This card allows users to update key personal details.

#### First Name and Last Name

- **Fields**: `First Name *`, `Last Name *`
- These are required fields that define how your name appears across the platform.

**To update:**
1. Click on the respective field.
2. Enter your updated first or last name.
3. Click the "Save changes" button when it becomes active.

#### Work Mail

- This is a read-only field displaying your registered work email.
- Users cannot modify this email.

#### Contact Number (Optional)

- Users can optionally add a contact number.
- Includes a country code selector.
- Accepts numbers in international format.

**To update:**
1. Select your country code from the dropdown.
2. Enter your mobile or landline number.
3. Click the "Save changes" button to save it.

---

### 2. Profile Picture Placeholder

A default avatar with the first initial of the user's first name is shown. Upload functionality may be available in later versions.

---

### 3. Save Changes Button

- The button is disabled until a valid change is made to the editable fields.
- Becomes active when any value is modified.

---

### Additional Notes

- Use accurate information for better communication and traceability.
- Contact information is optional but recommended for critical alerts or support calls.
