## New Features

- **New Apps Added:**
    - [**SAP Business One**](https://docs.appse.ai/app-integrations/sap-b1): Automate financials, sales, and inventory for enterprise businesses.
        - **New Triggers:**
            - New business partners created
            - New items created
            - Business partners updated
            - Items price updated
            - Items updated
        - **New Actions:**
            - Get an Item by ItemCode
            - Update an item
            - Create a new Item
            - Get a BusinessPartner by EmailAddress
            - Create a new BusinessPartner
            - Update a BusinessPartner
    - [**Salesforce**](https://docs.appse.ai/app-integrations/salesforce/): A CRM platform for managing customer relationships and driving sales.
        - **New Triggers:**
            - New quotations created
            - Contacts updated
            - New contact created
            - New account created
            - Accounts updated
        - **New Actions:**
            - Create an account
            - Update an account
            - Create a contact
            - Update a contact
    - [**OpenAI**](https://docs.appse.ai/app-integrations/openai/): A suite of AI tools for natural language processing and decision-making.
        - **New Action:**
            - Create chat completions
    - [**Shopify**](https://docs.appse.ai/app-integrations/shopify): An e-commerce platform helping businesses manage online stores & automate sales processes.
        - **New Triggers:**
            - New orders created
            - New customer created
            - Customers updated
        - **New Actions:**
            - Create a customer
            - Publish a product
            - Create product
            - Update product variants

- [**Credentials**](https://docs.appse.ai/platform/key-concepts/credentials/): Easily connect and manage SaaS application credentials from a centralized dashboard, enable secure and streamlined authentication for all integrations.
- [**Workflows**](https://docs.appse.ai/platform/key-concepts/workflow/): Build and automate business logic with a powerful workflow engine, create predefined actions, map data, and more.
- [**Nodes**](https://docs.appse.ai/platform/key-concepts/nodes/): Define the building blocks of your workflows, performing specific actions or transformations
- [**Trigger Nodes**](https://docs.appse.ai/platform/key-concepts/nodes/trigger/): Initiate a flow of data
    - [**App Triggers**](https://docs.appse.ai/platform/key-concepts/nodes/trigger/app-trigger): Starts workflows when specific events occur within an app
    - [**Manual Trigger**](https://docs.appse.ai/platform/key-concepts/nodes/trigger/manual): Starts a workflow manually
    - [**On-Schedule**](https://docs.appse.ai/platform/key-concepts/nodes/trigger/on-schedule): Triggers workflows based on a specified time schedule
    - [**Webhook**](https://docs.appse.ai/platform/key-concepts/nodes/trigger/webhook): Initiates workflows based on incoming HTTP requests

- [**Action Nodes**](https://docs.appse.ai/platform/key-concepts/nodes/built-in/) - Handle logical flow, data manipulation, and transformation
    - [**Merge**](https://docs.appse.ai/platform/key-concepts/nodes/built-in/merge): Combines multiple inputs into one output
    - **Splitter**: Divides data based on fields
    - **JSON Converter**: Converts JSON data into string for further processing
    - **Base64 Decode**: Decodes Base64-encoded data into JSON
- [**Rename Workflow**](https://docs.appse.ai/platform/key-concepts/workflow/introduction/#1-workflow-name): You can rename workflows to any name you desire now
- [**Save Button**](https://docs.appse.ai/platform/key-concepts/workflow/introduction/#2-save): Save your workflow building progress with just a click.
- [**Run Once**](https://docs.appse.ai/platform/key-concepts/workflow/introduction/#3-run-once): Execute your workflows at your will whenever you need.
- [**Active/Inactive Toggle**](https://docs.appse.ai/platform/key-concepts/workflow/introduction/#4-inactiveactive-toggle): Set your workflow to execute autonomously.
- [**Undo/Redo**](https://docs.appse.ai/platform/key-concepts/workflow/introduction/#4-inactiveactive-toggle): Revert or repeat actions
- [**Zoom In/Zoom Out**](https://docs.appse.ai/platform/key-concepts/workflow/introduction/#14-zoom-in): Adjust the workspace view
- [**Fit to Screen**](https://docs.appse.ai/platform/key-concepts/workflow/introduction/#16-fit-to-screen): Automatically resize the workflow canvas to fit the screen
- [**Tidy Up**](https://docs.appse.ai/platform/key-concepts/workflow/introduction/#17-tidy-up): Automatically organize the layout of your workflow
- [**Import/Export Workflows**](https://docs.appse.ai/platform/key-concepts/additional-options/import-export): Download your workflows to your local device and share across other organizations.
- [**Execution History**](https://docs.appse.ai/platform/key-concepts/execution-history): Track the execution history of your workflows, view data flow through each node, monitor performance and access execution metrics for troubleshooting.
- **Appearance**:  Toggle between themes like dark, light and system default based on preference
