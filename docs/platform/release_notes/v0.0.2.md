# appse ai - v0.0.2

**Release Date:** December, 2025

This major release for appse ai is marked by the introduction of **tiered subscriptions**, offering access to **premium features and advanced functionalities** across the platform. This new model is built alongside powerful additions to workflow design, including the essential new **HTTP Node** for external API integration, and powerful conditional logic using the **Filter Node** and **Decision Nodes**. We have also significantly expanded our ecosystem, rolling out **full integration support for 11 new apps**.

Platform usability and stability are dramatically improved with key enhancements: better setup control via Prefix/Suffix options in **Credential Management**, streamlined building with **Action Search**, and enhanced reliability in complex processes through improved **Execution Ordering**. Coupled with an updated Workflow Designer and over 100 individual fixes, this release ensures a more **powerful, reliable, and user-friendly** automation experience.

### New Features

- **New Built-in Nodes**:
    - **HTTP Node**: Make HTTP requests to any external API or service with various authentication methods
    - **Filter Node**: Filter data based on configured rules to conditionally process workflow data
    - **Decision Nodes**: Create conditional logic branches to route data based on evaluation criteria
- **Subscription**: Introduced tiered subscriptions to offer users access to premium features and functionalities across the platform.
    - **Smoother Upgrades**: One-click upgrade process for features and plans
    - **Secure Payments**: Accept credit cards and manage subscriptions

- **Credential Management**: Prefix and Suffix options for relavant credential properties
- **Execution Ordering**: Workflow execution ordering introduced for branched and multiple workflow scenarios
- **Action Search**: Introduced search functionality in Action List Dropdown
