This guide explains the features of the **Organization Settings** page in APPSeCONNECT and how users can manage organization members, roles, and statuses.

---

## Where to Find This

Navigate through the left sidebar: `Settings` > `Organization`

---

## Page Overview

The Organization tab displays all the members associated with your APPSe AI workspace. It allows `Admin users` to manage roles, track invitation statuses, and invite new members.

---

## Features and Descriptions

### 1. Organization Name

At the top of the section, the current organization name is displayed.

---

### 2. Members Table

The table contains the following columns:

#### Name
Displays the full name and email address of each user.

- Your account is marked with **"(You)"** next to your name.
- Each user entry provides a quick way to verify user identity via email.

#### Role
Indicates the assigned user role within the organization.

- **Admin**: Full access to manage settings, users, and integrations.
- **Editor**: Can create and modify items but may not manage users.
- **Viewer**: Read-only access across the platform.

#### Status
Displays the user's current status in the organization.

- **Active**: User has accepted the invitation and can access the platform.
- **Pending**: Invitation link has been sent to a new user in the organization.
- **Invitation Expired**: The invite link sent to the user has expired. The admin can re-invite the user.

#### Actions
An ellipsis (`...`) menu appears for each user (except for your own account) to perform further actions, such as:

- Change role
- Re-invite user
- Remove user

*Note*: Only Admin will be allowed to make these changes.
---

## Inviting Members

### Invite Members Button

Located at the top-right of the screen.

**To invite a new member:**
1. Click the **"Invite Members"** button.
2. Enter the user's email and assign a role(editor or viewer).
3. Send the invite.

The invited user will appear in the members list with a status like "Pending" or "Invitation Expired" depending on whether they've accepted the invite.
